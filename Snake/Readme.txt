I. File list
------------

AI.java				Contains the main part of the Artificial Intelligence
Controller.java			The Controller for this program
KeyboardListener.java		Implements keyboard listener in our program
Main.java			Driver for Snake
Apple.java			Apple generator
Audio.java			Audio implementation
Enumerators.java		Contains all our enumerators
Field.java			Our homemade grid elements
HighScore.java			High Score implementation
Model.java			Contains almost every method used to tie program together
Snake.java			Contains our Snake elements i.e. Head body and tail, but also methods to move the snake
ComboBoxRenderer.java		#############################################
GamePanel.java			The game field panel
OptionsMenu.java		The options panel
StartMenu.java			The start menu panel
View.java			The overall frame that ties the view together

B. Full list of available hotkeys
   
Movement:

	Up: 		W  or ArrowUP
	Down: 		S  or ArrowDOWN
	left:		A  or ArrowLEFT
	Right: 		D  or ArrowRIGHT

Other Hotkeys

	Pause: 		P
	New Game:	N
	Start Menu: 	Esc
	Options: 	O
	Theme 1: 	1
	Theme 2: 	2
	Artificial Int: 3
	Speed UP:	4
	Speed Down:	5
